# Extra Pastel Blocks

**Extra Pastel Blocks** is a Fabric Minecraft mod that adds a carefully curated set of clean, builder-friendly pastel blocks with matching stairs and slabs.  
The palette is designed for modern, cozy, and large-scale builds, focusing on calm, material-like colors that blend well with vanilla blocks.
